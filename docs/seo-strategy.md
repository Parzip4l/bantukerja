# SEO Strategy

## Positioning
- Domain: `bantukerja.online`
- Tagline: `Tools dan template gratis untuk kerja, bisnis, dan administrasi harian.`
- Target utama: long-tail keyword dengan intent informasional dan utilitas.

## Pilar SEO
- Tool pages: keyword intent tinggi, potensi CTR dan RPM baik.
- Template pages: keyword evergreen, kuat untuk internal linking.
- Blog pages: menangkap search intent edukasi dan mengalirkan traffic ke tool/template.

## Implementasi Teknis
- Title dan meta description dinamis.
- Canonical URL di setiap halaman.
- Open Graph dan Twitter card.
- JSON-LD untuk `WebSite`, `Article`, `FAQPage`, dan `BreadcrumbList`.
- Sitemap XML dan robots.txt.
- Konten utama SSR, bukan SPA.
- Slug SEO-friendly.
- Admin login diberi `noindex,nofollow`.

## Struktur Konten
- Home mengangkat kategori, tools populer, template populer, dan artikel terbaru.
- Halaman detail tool/template/blog saling terhubung lewat related content.
- Artikel diarahkan untuk menjawab pertanyaan pengguna sebelum mendorong klik ke tool/template.

## Keyword Cluster Awal
- THR: kalkulator thr, cara menghitung thr, thr proporsional.
- Gaji: kalkulator gaji bersih, estimasi take home pay, contoh potongan gaji.
- Lembur: hitung lembur sederhana, upah per jam, lembur hari libur.
- Dokumen: surat resign, surat izin kerja, surat kuasa, berita acara.
- Bisnis: invoice sederhana, perbedaan invoice dan kwitansi, template mou.

## Checklist On-Page
- H1 tunggal dan jelas.
- Paragraf pembuka ringkas dan sesuai search intent.
- FAQ untuk query turunan.
- Breadcrumb internal.
- Related content di sidebar/bagian bawah.
- Pagination untuk listing.

## Tahap Lanjutan
- Tambahkan featured image unik untuk artikel.
- Buat halaman kategori publik bila cluster konten sudah bertambah.
- Tambahkan analitik Search Console dan evaluasi query CTR rendah.
